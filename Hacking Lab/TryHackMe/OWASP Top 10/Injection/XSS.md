This is a Reflected XSS attack technique that triggers the execution of malicious code by injecting %0D%0A (line break) characters.<br>
반사형 XSS (Reflected XSS)으로 %0D%0A (줄바꿈)등을 넣어 악성 코드가 실행되게 유도하는 방식입니다.
(CVE-2023-38501)
